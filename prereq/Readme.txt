Dependencies 
- Requires PHP in the system environment.
- In Php.ini, intl, mbstring, mysqli extensions should be enabled.
- Requires XAMPP installed, running Both Apache and MySql Modules.
- Mysql username should be root, with no passwords, hostname localhost.


How to run
- Create a database named code_igniter, and import the code_igniter.sql inside.
- If you are having trouble configuring php.ini, I have included my configurations as well.
- In project directory type `php spark serve` open `localhost:8080` in any browser.
